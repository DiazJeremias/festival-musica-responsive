# Festival de Música - Sitio Web

Un sitio web estático y responsive desarrollado como examen parcial para la universidad, simulando la plataforma oficial de un festival de música con su grilla de artistas, información de locación y venta de entradas.

## 🎓 Contexto Académico

- **Institución:** Universidad Nacional de Hurlingham (UNAHUR)
- **Materia:** Taller de lenguajes de marcado y tecnologías web
- **Instancia:** Primer parcial.

## Tecnologías y Conceptos Aplicados

- **HTML5:** Estructuración semántica del sitio para SEO y accesibilidad.
- **CSS3 Puro:** Diseño visual, tipografías y maquetado utilizando Flexbox / CSS Grid.
- **Responsive Design:** Adaptabilidad completa para dispositivos móviles, tablets y escritorio mediante Media Queries.
- **Git:** Control de versiones durante el desarrollo del examen.
